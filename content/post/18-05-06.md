交互式选择项目模板
```
mvn archetype:generate
```
项目使用 SSH 框架可以查找 struts 2 相关的模板

打 war 包
```
mvn package 
```
war 包拷贝到 Tomcat app 目录下，服务器启动时自动解压部署。
